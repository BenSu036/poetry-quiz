# 古诗词储备量量化测试

一个极简风格的古诗词知识测试网页，通过自适应难度测试，精准量化你的诗词储备数量。

## 在线体验

👉 [点击开始测试](https://BenSu036.github.io/poetry-quiz/)

## 功能特点

- **6级难度体系**：从入门到大神，逐级挑战
- **自适应难度**：连续答对3题升级，答错维持当前难度
- **30题测试**：快速精准评估诗词储备量
- **动态选项生成**：每道题的干扰项字数与答案完全一致，具有诗意迷惑性
- **全局选项去重**：每个选项在整个测试过程中只出现一次
- **结果量化**：输出具体储备量区间（如 400-500首）

## 技术栈

纯前端实现：HTML + CSS + JavaScript，无需后端服务器。

## 本地运行

下载项目后，直接用浏览器打开 `index.html` 即可。

## 项目结构

```
├── index.html      # 页面结构
├── styles.css      # 样式（中国风配色）
├── app.js          # 核心逻辑（自适应难度、结果计算）
└── questions.js    # 题库（168首诗词，6级难度）
```
